## 🎯 Project Objective

To develop a **multi-agent AI simulation of a business ecosystem** where AI agents role-play as companies, consumers, countries, suppliers, and regulators. The simulation models how these entities respond to **external shocks** and generates **pre-emptive strategic insights**.

## 🏗️ Agent Architecture

### Agent Roles Defined
Each agent is assigned:
- **Role** (e.g., company, consumer, regulator, supplier, country)
- **Objective** (profit maximisation, market protection, cost minimisation, etc.)
- **Memory** (history of past decisions and shock events)
- **Decision Rules** (how it responds to changes in the ecosystem)

### Agent Types in the Simulation
1. **Company Agents** — e.g., Tata Steel, Maruti Suzuki
2. **Consumer Agents** — demand-side behavioural responses
3. **Country/Government Agents** — policy, tariff, geopolitical decisions
4. **Supplier Agents** — raw material pricing, availability shocks
5. **Regulator Agents** — antitrust, compliance, intervention decisions

---

## ⚡ External Shocks Modelled

The simulation is designed to respond to the following shock types:
- Wars / geopolitical disruptions
- New market entrants (competitive disruption)
- Cartel / nexus formation
- Corporate fraud events
- Regulatory interventions
- Demonetisation-type macro shocks
- Raw material price volatility
